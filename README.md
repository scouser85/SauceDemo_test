Тестовый проект автоматизации тестирования логина сайта Saucedemo.com
В проекте реализовано 5 тестов для сценариев логина:

1. Успешный логин (название теста Successfull_Login)
2. Логин с неверным паролем (название теста LoginWithWrongPassword)
3. Логин заблокированного пользователя (название теста LockedOutUserLogin)
4. Логин с пустыми полями (название теста LoginWithEmptyCredentials)
5. Логин пользователем performance_glitch_user - для проверки успешного логина при различных задержках (название теста PerformanceGlitchLogin)

Язык программирования: Java

Фреймворки и библиотеки: Selenium WebDriver, TestNG

Отчетность: Allure Reports

Сборщик проектов: Maven

Паттерн проектирования: Page Object Model (POM)
